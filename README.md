DESKRIPSI APLIKASI

UniLink adalah aplikasi berbasis Android yang dibangun sebagai platform digital untuk mengintegrasikan kebutuhan Program Magang antara mahasiswa dan perusahaan mitra. Mengadopsi bahasa desain modern bergaya Bento UI, aplikasi ini bertujuan untuk menghilangkan hambatan informasi dalam mencari peluang profesional di dunia kerja. UniLink memfasilitasi mahasiswa dalam menemukan posisi magang yang relevan dengan kompetensi mereka, sekaligus membantu perusahaan dalam menjaring talenta muda yang tepat secara efisien.

DAFTAR DAN DESKRIPSI FITUR

- Internship Explorer (Connect)

Menampilkan daftar perusahaan yang membuka lowongan magang dalam bentuk kartu profil (Bento Grid) yang informatif dan mudah dipahami.

- Profil Perusahaan & Mahasiswa

Menyediakan detail informasi mengenai profil mitra korporat dan resume digital mahasiswa untuk mempermudah proses pencocokan kualifikasi magang.

- Pencarian & Filter Terintegrasi

Memungkinkan pengguna untuk menyaring peluang magang berdasarkan kategori industri atau minat tertentu untuk mempercepat proses penemuan kemitraan.

- Sistem Notifikasi (Alerts)

Memberikan pembaruan waktu nyata terkait status aplikasi magang, pesan dari mitra perusahaan, atau pengumuman program baru.

- Navigasi Dinamis

Antarmuka navigasi bawah yang responsif untuk perpindahan cepat antar menu utama seperti Beranda, Event, Tambah Postingan, Koneksi Kemitraan, dan Notifikasi.


SPESIFIKASI TEKNIS

- Bahasa: Kotlin.

- Layout: XML dengan Material Design 3 Components (MaterialCardView, ShapeableImageView).

- Versi SDK: Target API 34+.

- Arsitektur: Menggunakan sistem Fragment Management dengan Bottom Navigation untuk performa yang ringan.
